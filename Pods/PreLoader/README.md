##Usage:
```objc
PreLoader *preLoader = [[PreLoader alloc] initWithFrame:CGRectMake(60, 300, 250, 250)
                                                      color:spotColor
                                            backgroundColor:[UIColor clearColor]];
                                            
[self.view addSubview:preLoader];
```  

<br>
##CocoaPods:  
```
pod 'PreLoader', '~> 0.3.1'
```
